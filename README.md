# Robe Market

A mobile-first marketplace starter app built with Next.js and Tailwind CSS. It includes:

- Product catalog with search and filters
- Shopping cart and wishlist
- User registration and login flow
- Admin inventory dashboard
- Order placement with Cash on Delivery and local payment placeholders
- Contact and location support

## Setup

1. Open a terminal in `robe-market`
2. Run `npm install`
3. Run `npm run dev`
4. Visit `robe market`.
